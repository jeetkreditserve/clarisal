# Framer Motion Skill for Claude Code

Production-grade animation patterns for React and Next.js using Motion (formerly Framer Motion).

## What's included

- Correct imports (`motion/react`, `motion/react-client`, `framer-motion`)
- Core patterns: `motion.*` components, MotionValues, transitions
- Scroll animations: `whileInView`, `useScroll`, `useSpring`
- Exit animations: `AnimatePresence`
- Stagger with `variants`
- Layout animations / `layoutId`
- Gesture animations (hover, tap, drag)
- Accessibility: `useReducedMotion`
- Performance rules
- Recipes: progress bar, counter, page transition, gradient shimmer

## Installation

```bash
claude plugin marketplace add Schoepplake/framer-motion-skill
claude plugin install framer-motion@framer-motion-skill
```
